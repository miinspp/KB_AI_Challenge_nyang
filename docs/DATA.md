# 데이터 범위 및 출처 정리

> 마지막 갱신: 2026-07-20 (데이터 기준 분기: **2025 Q2 ~ 2026 Q1**)
> 한 줄 요약: **매출 분포는 "서울 한정" 추정 통계, 이익률 벤치마크는 "전국" 조사 통계.** 홈택스 등 개별 점포의 신고·실측 데이터는 포함되지 않는다.

---

## 1. 지역 범위 (가장 중요)

| 산출 요소 | 지역 범위 | 왜 |
|---|---|---|
| 매출 백분위 분포 (상위 % 산출의 핵심) | **서울 한정** | 서울시 상권분석서비스 데이터 (서울 상권 1,576개) |
| 이익률 벤치마크 (순수익·비용효율 계산) | **전국** | 소상공인실태조사는 전국 표본조사 |

⚠️ 따라서 서비스가 말할 수 있는 것은 **"서울 △△업종 (추정매출 기준) 상위 X%"** 이다.
"전국 상위 X%" 라는 표현은 현재 데이터로는 근거가 없으므로 사용 금지.

## 2. 데이터 출처 (2개 공공 소스)

### ① 서울시 상권분석서비스 「추정매출-상권」 — 분포의 본체
- 제공: 서울 열린데이터광장 [OA-15572](https://data.seoul.go.kr/dataList/OA-15572/S/1/datasetView.do) Open API (무료)
- 성격: 서울시가 **카드사 결제 데이터 등 빅데이터로 "추정"** 하여 상권 단위로 집계·공개한 통계
  - 개별 가게 식별 정보 없음. 최소 단위 = **상권 × 업종** (개인정보·과세정보 아님)
  - 점포주가 신고/입력한 값이 아님 → **홈택스와 무관**
- 단위: 분기 × 상권(1,576개: 골목상권·발달상권·전통시장·관광특구) × 업종
- 주요 필드: 분기 추정매출액(`THSMON_SELNG_AMT`), 점포수(`STOR_CO`, 별도 「점포-상권」 API 조인)

### ② 소상공인실태조사 (통계청·중소벤처기업부) — 이익률 벤치마크
- 전국 조사. 업종별: 2022년 확정 / 전체 평균: 2023년 잠정 ([kosis.kr](https://kosis.kr))
- 사용 값: 업종 그룹별 **평균 영업이익률** (`pipeline/benchmarks.json`)

| 그룹 | 평균 영업이익률 |
|---|---|
| 숙박·음식점업 | 22.1% |
| 도·소매업 | 10.6% |
| 교육서비스업 | 35.7% |
| 예술·스포츠·여가업 | 23.1% |
| 수리·기타 서비스업 | 30.2% |
| 소상공인 전체 평균 | 12.6% |

## 4. 가공 흐름

```
서울 Open API (JSON 응답, ~206만 건 전량 수집)
  → pipeline/raw/*.csv               (원본 보관, git 미추적)
  → 점포당 월매출 = 분기 추정매출 ÷ 3 ÷ 점포수
  → 상권×업종 관측치를 점포수로 가중한 업종별 백분위(0~100) 격자
     (+ 상권유형[골목/발달/전통시장]별 보조 격자 — 표본 상권 30개 이상 유형만)
  → backend/src/main/resources/data/industry_distributions.json  (DB 없음, 기동 시 메모리 로드)
```

재수집·갱신 명령은 `pipeline/collect_seoul_sales.py` 상단 주석과 README 참고.

## 4-1. 비용구조·임대료 벤치마크 (산출식 v2 보정축 근거)

> 산출물: `backend/src/main/resources/data/public_benchmarks.json`
> 수집 스크립트: `pipeline/collect_public_benchmarks.py`
> 용도: 비용 구조 축을 "10% 경험칙"에서 **업종별 실측 평균 대비**로 승격 + 리포트 비교 표기.

| 데이터 | 출처 / 표ID | 기준 기간 | 수집 방식 | 서비스 사용처 |
|---|---|---|---|---|
| 업종별 영업비용 비목 (매출원가·급여·임차료·기타) | 소상공인실태조사 `DT_3ME0144` (중기부·KOSIS) | **2023년** (기업체당 연간) | 스냅샷¹ | 원가율·인건비율·**임차료율** 업종 벤치마크 → 비용구조 점수 |
| 업종별 기업체당 보증금·월세 | 소상공인실태조사 `DT_3ME0132` | **2024년** | 스냅샷¹ | 평균 월세 참고 표기 |
| 업종별 매출액·영업이익 | 중기부 보도자료(2023 잠정, 2025-02 발표) | **2023년** | 스냅샷 | 이익률·비목 비율 분모(매출), 항등식 검증 |
| 서울 상가 임대료 (소규모·중대형·집합) | 한국부동산원 임대동향조사 R-ONE `T248223…`/`T244363…`/`T244913…` | **2024 Q3 ~ 최신** (분기) | **라이브 API**² | 서울 상권별 ㎡당 임대료 컨텍스트 |

¹ **스냅샷 사유**: KOSIS 뷰어의 내부 조회 POST(`html.do`)가 세션 토큰에 의존해 무인증 재현이 불안정.
  연 1회 갱신되는 소량(업종 대분류 12행) 통계라, 뷰어에서 실제 조회한 값을 표ID·기준연도·추출일(스크립트 `KOSIS_SNAPSHOT_DATE`)과 함께 상수로 고정하고,
  **매출 − 영업이익 == 영업비용** 항등식으로 두 출처(KOSIS·보도자료) 정합성을 자동 검증한다(불일치 시 수집 실패).
² **라이브 사유**: R-ONE 뷰어의 JSON 엔드포인트(`sttsDataPreviewList.do`)는 무인증·무세션으로 응답 → 실행 시점 최신 분기를 그대로 수집.

### 상권유형별 임차료율 지역 보정 (한국부동산원 서울 상가 임대료)

임차료율 벤치마크(전국 평균)는 서울 상권의 임대료 수준을 반영하지 못한다. 그래서
한국부동산원 R-ONE **서울 상가유형별 ㎡ 임대료**를 상권유형에 대응시켜 **완만한 보정 배수**를 만든다.

| 상권유형 (서울시 상권분석) | ↔ R-ONE 상가유형 | 서울 임대료(천원/㎡)* | 보정 배수 |
|---|---|---|---|
| 골목상권 (가로변 소형) | 소규모상가 | 52.5 | **×1.00** (기준) |
| 발달상권 (번화가 중대형) | 중대형상가 | 56.4 | **×1.074** |
| 전통시장 (시장 내 구분) | 집합상가 | 49.6 | **×0.945** |

\* 최신 분기(2026 Q1) 서울 평균. 배수 = 각 상가유형 ÷ 소규모상가.

- **적용**: `조정 임차료율 벤치마크 = 전국 임차료율 × 보정배수`. 상권유형을 고른 경우에만 적용하며,
  "서울 전체"(미선택)에는 전국 벤치마크를 그대로 쓴다.
- **감쇠 없음 근거**: ㎡ 임대료와 임차료율(임차료÷매출)은 1:1 비례하지 않는다(번화가는 매출도 높음).
  상가유형 간 실측 격차 자체가 이미 ±10% 내로 완만하므로 임의 감쇠 계수 없이 그대로 쓴다.
- **효과 예**: 임대료 부담률 12%인 한식집 → 발달상권 39.8점(관대) · 골목 35.3점 · 전통시장 31.5점(엄격).
- **한계**: R-ONE 지역 구분(도심/강남/…)과 상권유형(골목/발달/전통)은 taxonomy가 달라, 상가유형(소규모/중대형/집합)을
  프록시로 삼은 근사 대응이다(스크립트 `AREATYPE_SHOPTYPE`). 매출 분포는 서울, 비용 벤치마크는 전국 기준이라는 지역 혼재는 유지된다.

> 산출식 전체 흐름·가중치는 [`METHODOLOGY.md`](METHODOLOGY.md) 참고.

## 5. 데이터가 "말할 수 없는 것" (한계)

1. **전국 순위** — 분포가 서울 한정. 전국 확장 시 상권정보시스템(소진공) 등 별도 소스 필요.
2. **실측 순수익 분포** — 타 점포의 순이익·원가율은 세법상 보호 대상이라 어떤 경로로도 수집 불가.
   현재는 매출 분포 × 업종 평균 이익률로 **유도한 근사 분포**를 사용 (점포별 이익률 편차 미반영, API 응답 notes에 명시).
3. **개별 점포 정확 매출** — 원천이 카드 데이터 기반 "추정치"이며 상권 단위 집계임.
4. 본인 데이터(매출·지출)는 사용자 직접 입력에 의존. 향후 홈택스 본인 동의 연동(삼쩜삼 방식)으로 대체 가능.
